## 4. 반응형 고려사항

### Extension
- 고정 크기: 320px x 400px (크롬 팝업 권장 크기)

### Web App
| 브레이크포인트 | 레이아웃 |
|---------------|---------|
| Desktop (≥1024px) | 사이드바 + 메인 콘텐츠 |
| Tablet (768-1023px) | 축소된 사이드바 |
| Mobile (≤767px) | 하단 네비게이션, 단일 컬럼 |

### 마인드맵
- 터치 제스처 지원 (핀치 줌, 드래그)
- 최소 터치 타겟: 44px x 44px
