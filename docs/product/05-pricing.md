- **Pro**: 파워 유저, 프리랜서
- **Enterprise**: 팀/기업

## AI 토큰 계산 기준

MindHit은 AI 기능 사용 시 토큰을 소비합니다.

| 작업 | 예상 토큰 소비 |
|-----|---------------|
| 페이지 요약 (1페이지) | ~500 토큰 |
| 키워드 추출 (1페이지) | ~200 토큰 |
| 마인드맵 생성 (10페이지 세션) | ~3,000 토큰 |
| 마인드맵 생성 (50페이지 세션) | ~10,000 토큰 |

### 플랜별 예상 사용량

| 플랜 | 토큰 한도 | 예상 세션 수/월 |
|-----|----------|----------------|
| Free | 50,000 | 10-15개 세션 |
| Pro | 500,000 | 100-150개 세션 |
| Enterprise | 무제한 | 무제한 |

*평균 30페이지/세션 기준*

### 토큰 측정 방식

토큰 사용량은 **AI API 응답에서 직접 제공**되므로 100% 정확합니다.

**측정 원리:**

| AI 제공업체 | 응답 필드 | 설명 |
|------------|----------|------|
| OpenAI | `response.usage.total_tokens` | prompt + completion 토큰 합계 |
| Google Gemini | `response.usageMetadata.totalTokenCount` | 총 토큰 수 |
| Anthropic Claude | `response.usage.input_tokens + output_tokens` | 입출력 토큰 합계 |

**측정 흐름:**

```text
1. 사용자가 마인드맵 생성 요청
2. 서버가 AI API 호출
3. AI API 응답에 실제 사용 토큰 포함
   └── { "usage": { "total_tokens": 3847 } }
4. 서버가 token_usage 테이블에 기록
   └── INSERT INTO token_usage (user_id, tokens_used, period_start, ...)
5. 사용자 대시보드에 반영
```

**정확도 보장:**
- 예상치가 아닌 **실제 사용량** 기록
- AI API가 반환하는 값을 그대로 저장
- 사용자는 정확한 토큰 소비량 확인 가능

## 빌링 주기 (Billing Period)

사용자마다 **빌링 주기가 다르게** 설정됩니다.

### 플랜별 빌링 주기

**모든 플랜은 가입일/구독일 기준으로 30일 주기**로 토큰이 리셋됩니다.

| 플랜 | 빌링 주기 시작 | 빌링 주기 종료 | 리셋 시점 |
|-----|--------------|--------------|----------|
| **Free** | 가입일 | 가입일 + 30일 | 30일마다 자동 리셋 |
| **Pro** | 구독 시작일 | 다음 결제일 전날 | 결제일에 자동 리셋 |
| **Enterprise** | 계약 시작일 | 계약 갱신일 전날 | 갱신일에 자동 리셋 |

### 빌링 주기 예시

```
Free 플랜 사용자 A (12월 10일 가입):
├── 빌링 주기: 가입일 기준 30일
├── 1차 기간: 12/10 ~ 1/8 (50,000 토큰)
├── 2차 기간: 1/9 ~ 2/7 (50,000 토큰 리셋)
└── 3차 기간: 2/8 ~ 3/9 (50,000 토큰 리셋)

Free 플랜 사용자 B (12월 25일 가입):
├── 빌링 주기: 가입일 기준 30일
├── 1차 기간: 12/25 ~ 1/23 (50,000 토큰)
└── 2차 기간: 1/24 ~ 2/22 (50,000 토큰 리셋)

Pro 플랜 사용자 C (12월 15일 구독 시작):
├── 빌링 주기: 구독 시작일 기준
├── 1차 기간: 12/15 ~ 1/14 (500,000 토큰)
├── 2차 기간: 1/15 ~ 2/14 (500,000 토큰 리셋)
└── 결제: 매월 15일 자동 결제
```

### 업그레이드/다운그레이드 시 빌링 주기

| 변경 | 빌링 주기 처리 |
|-----|--------------|
| **Free → Pro** | 업그레이드 시점부터 새 빌링 주기 시작, 토큰 즉시 500,000으로 증가 |
| **Pro → Free** | 현재 결제 기간 끝까지 Pro 유지, 다음 기간부터 Free (가입일 기준 30일 주기) |

## FAQ

### Q: Free 플랜에서 Pro로 업그레이드하면 기존 세션은 어떻게 되나요?
A: 모든 세션이 유지되며, 30일 제한이 해제되어 무제한 보관됩니다.

### Q: 토큰이 부족하면 기록 중인 세션은 어떻게 되나요?
A: 세션 기록은 계속됩니다. 마인드맵 생성만 토큰 충전/업그레이드 후 가능합니다.

### Q: 팀원마다 별도 결제가 필요한가요?
A: Enterprise 플랜은 팀 단위 요금제입니다. 팀원 수에 따라 가격이 조정됩니다.

### Q: 환불 정책은 어떻게 되나요?
A: 결제 후 7일 이내 전액 환불 가능합니다. 그 이후는 환불되지 않으며 남은 기간 동안 사용 가능합니다.

### Q: API 접근은 어떤 플랜에서 가능한가요?
A: Enterprise 플랜에서만 API 접근이 가능합니다.
